## Technologies:

* Ruby 3.0.0
* Rails ~> 7.0.2", ">= 7.0.2.3"


### Setup

Navigate to directory where the code script is located

```sh
$ cd ~/minesweeper
```
Install the dependencies and devDependencies.

```sh
$ rvm use 3.0.0
````

```
$ bundle install
```

### Steps to run code


```sh
$ rake db:setup
```

```sh
$ rails s or foreman start
```
